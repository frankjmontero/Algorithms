# **Get N Fibonacci Number**

Find the fibonacci number 'n' in the fibonacci sequence.

## **Testing**

Run instructions:

In Javascript
```
node NFibonacciNumber.js
```

Testing values
```js
console.log(fibSec(1) === 1);
console.log(fibSec(5) === 5);
console.log(fibSec(7) === 13);
console.log(fibSec(8) === 21);
```
